# Bookstore API Automation (Java + RestAssured)

## 🚀 How to Run
1. Start the FastAPI server from provided project:
   ```bash
   uvicorn main:app --reload
   ```
2. Run tests with Maven:
   ```bash
   mvn clean test
   ```

## 📊 Reporting
- Extent report will be generated in `reports/` folder after execution.

## ⚙️ Tech Stack
- Java 11+
- RestAssured
- TestNG
- Extent Reports
- Maven
